# Модуль веб-интерфейса (Front UI)

Микросервис **Front UI** представляет собой пользовательский веб-интерфейс банковского приложения. Он обеспечивает визуальное взаимодействие пользователя с системой, позволяя просматривать баланс, управлять профилем и совершать финансовые операции в удобном графическом виде.

### Основные функции:
* **Личный кабинет:** Визуализация данных профиля (ФИО, возраст) и текущего остатка на счете.
* **Формы операций:** Интерфейсы для ввода сумм при пополнении, снятии и выполнении переводов.
* **История операций:** Отображение списка уведомлений и транзакций пользователя в читаемом виде.
* **Авторизация:** Интеграция с Keycloak для входа в систему и выхода (Logout).

### Технологический стек:
* **Spring WebFlux:** Реактивный движок для обработки веб-запросов.
* **Thymeleaf:** Серверный шаблонизатор для динамического формирования HTML-страниц.
* **Spring Security:** Реализация паттерна **BFF (Backend For Frontend)** для безопасного хранения сессий и работы с OAuth2/OIDC.
* **Bootstrap:** CSS-фреймворк для адаптивной и быстрой верстки интерфейса.

### Логика работы:
1. **Аутентификация:** При попытке доступа к защищенным страницам пользователь перенаправляется на страницу логина Keycloak.
2. **Взаимодействие с API:** После входа фронтенд делает запросы к микросервисам через **API Gateway**.
3. **Token Management:** Сервис автоматически управляет обновлением Access токенов и передает их в заголовках при обращении к бэкенду.

### Основные страницы:
* `/main` — Главная страница с информацией о системе.
* `/account` — Личный кабинет (данные из сервиса Accounts).
* `/cash` — Панель управления наличностью (сервисы Cash).
* `/transfer` — Панель управления переводами (сервис Transfer).

### Интеграция
Сервис настроен как **OAuth2 Client**. Конфигурация клиента (Client ID, Secret) должна совпадать с настройками в Keycloak, указанными в файле `.env`.

### Разработка и запуск
Сборка только модуля Front UI локально:
```bash

./gradlew :front-ui:bootJar
```
Для локального запуска (вне Docker) требуются активные сервисы Keycloak и Gateway, так как UI является лишь визуальной оболочкой над API.

### Запуск тестов
Модуль содержит тесты контроллеров и проверку корректности отображения шаблонов:
```bash

./gradlew :front-ui:test
```
